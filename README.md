# react-book-store
基于vite搭建的整合React+Mobx+TypeScript+Antd图书管理项目
### 获取源码
``git clone https://github.com/Lxy0707s/react-book-store.git``
### 初始化项目
``yarn install``
### 项目运行
``yarn run dev``
